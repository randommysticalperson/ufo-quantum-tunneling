## Overview

This repository contains a complete, mathematically rigorous speculative physics analysis of a hypothetical UFO using quantum tunneling (or classical kinetic penetration) to pass straight through the Earth. The analysis uses the **full WKB (Wentzel–Kramers–Brillouin) approximation** with a **spatially varying potential barrier** derived from the real **PREM (Preliminary Reference Earth Model)** density profile.

## Physics Summary

### WKB Approximation
The transmission coefficient is:

```
T ≈ exp(−2Γ),   Γ = ∫ κ(x) dx
κ(x) = √(2M[V(x) − Eₖ/L]) / ℏ
```

where `V(x)` is derived from the PREM density profile and integrated numerically over 100,000 path points.

### Cherenkov Radiation
At v = 0.90c, the craft exceeds c/n in both desert sand (n=1.46) and ocean water (n=1.34), generating Cherenkov radiation cones at 40.4° and 34.0° respectively.

### Ionization (Bethe-Bloch)
Energy loss through each Earth layer is calculated using the relativistic Bethe-Bloch formula, peaking in the dense metallic cores.
